1. В курсовой работе передо мной стояла задача выполнить верстку веб-сайта по представленному макету и загрузить ее на свой
 репозиторий в GitHub.
2. Для реализации поставленной задачи мною были использованы редактор кода VSCode, локальный сервер для тестировки и отладки
на своем компьютере, язык гипертекстовой разметки HTML, формальный язык описания внешнего вида веб-страницы CSS (каскадные страницы 
стилей), также мною были использованы навыки работы с веб-сервисом для хостинга GitHub, графическим редактором Figma.
 В качестве исходных данных у меня был Figma-макет.
3. В результате был сверстан лендинг и загружен на сервис GitHub Pages.
Cсылку к нему прилагаю https://mariackopkova.github.io/

Вначале я создавала базовую файловую структуру проекта: index.html CSS img fonts js. С помощью тега link подкючила к index.html
файл стилей, иконки, фавиконки.
Cкопировала макет с Figma.
Проанализировав макет, определила сетку, макет тегов страницы.
основные теги header,main,section,footer.
Для навигации использован тег nav.
Для подключения файла JS использовала тег script, который вставила в конце, перед закрывающим тегом </body>.
Выявила и создала  базовые переменные в файле сss: с помощью @font-face семейства шрифтов,
c помощью псевдокласса root основной цвет, размер шрифта, также использовала псевдоклассы before и after.
Для содания адаптивного сайта использовано правило  @media - медиа-запросы,с помощью которых можно менять 
стиль элементов, отталкиваясь от устройств, на которых будут просматривать веб-страницу.
