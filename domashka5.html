Теги H1-h6 используются для написания заголвков del - использеются для выделения текста,который был удален в новой версии документа.
ins - используется для выделения текста, который был добавлен в новой версии документа.
small - уменьшает размер шрифта на единицу по сравнению с обычным текстом.
b - устанавливает жирноетначертание текста
strong - предназначен для акцентирования текста
sub - Отображает шрифт в виде нижнего индекса.
sup - Отображает шрифт в виде верхнего индекса
nav - задает навигацию по сайту
aside - Определяет блок сбоку от контента для размещения рубрик, ссылок на архив, меток и другой информации
main - предназначен для основного контента
footer - задаёт «подвал» сайта или раздела, в нём может располагаться имя автора, дата документа, контактная и правовая информация.
section - Задаёт раздел документа, может применяться для блока новостей, контактной информации, глав текста, вкладок в диалоговом окне и др. Раздел обычно содержит заголовок.
