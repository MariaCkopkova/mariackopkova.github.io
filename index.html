p- определяет место для параграфа. Обязателен закрывающий тег.

br - строчный тег, говорит браузеру, что в этом месте нужно перенести текст на новую строку.

span - определяет строчный элемент документа. Обязателен закрывающий тег.
Mожно использовать для форматирования отрывков текста.Можно, например, выделить заглавную букву красным цветом.
К нему применимы только глобальные атрибуты.Например тег class и id

    оl - тег устанавливает нумерованный список. Каждый элемент списка должен начинаться с тега "li".
    Тег li определяет отдельный элемент списка. Внешний тег "ol" говорит, что этот тег нумерованный.Обязателен закрывающий тег
    Атрибуты тега :

  1. type - усианавливает вид маркера списка
  2. resersed - нумерация в списке становится по убыванию.
  3. start - задает число с которого будет начинаться список.

    тег "ul" - устанавливает маркированный список. Каждый элемент списка должен начинаться с тега "li".Обязателен закрывающий тег.

dl - блочный тег, создающий список определений.В него входят элементы dt и dd

dt
-создает термин.
dd

a - используется для создания ссылки либо на сайт либо на место на стринце

тег img - используется для отображения на веб-странице изображений в графическом формате GIF, JPEG и PNG.

blockquote - используется для оформления цитаты

address - предназначен для хранения информации об авторе и может включать в себя любые элементы HtmL, вроде ссылок, текста и т.д.

code - строчный элемент.предназначен для отображения программного кода. 
pre - блочный тег, определяет предварительно форматированный текст.
hr - блочный элемент, рисует горизонтальную линию.
